7) Local M3U8
  • 6.1) Paste your m3u8 URL inside text file. The URL must be a valid link, such as https://test-streams.mux.dev/x36xhzz/x36xhzz.m3u8
  • 6.2) Rename the text file by your preferred filename, such as Big Buck Bunny.iptv
  • 6.3) Copy your text file(Big Buck Bunny.iptv) to internal or external storage
  • 6.4) Goto Menu > Local M3U8 to play your local m3u8 files. Click Refresh to reload all local m3u8 files inside local storages
  • 6.5) The file extension is iptv
  • 6.6) Each file must contain only single m3u8 URL
7) Goto Menu > Clear Local Caches to clear caches on local storages
